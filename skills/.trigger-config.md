## Uso

El agente detectará automáticamente qué skill cargar basándose en las palabras clave de tu mensaje.

Para cargar manualmente:
```
/skill <nombre-del-skill>
```

Ejemplo:
```
/skill dxrk-react-patterns
/skill dxrk-testing-strategy
```
